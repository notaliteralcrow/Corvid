CORVID 2.0 — INSTALLATION
=========================

QUICK INSTALL (recommended)
-----------------------------
Run each installer in Code Editor, in order.
Save, build, and run each one before starting the next.

  installer0.src   — core libs and facades
  installer1.src   — managers: database, intel, attack, bounce
  installer2.src   — managers: roost, platform, shinylibs, UI
  installer3.src   — commands_1 and commands_2
  installer4.src   — commands_3 and commands_4
  installer5.src   — commands_bounces and commands_5
  installer6.src   — data: chainsaw and pregens (part 1)
  installer7.src   — pregens (part 2) + main.txt + builds binary

When installer7 completes: "Build successful! Run /Corvid/corvid"


MANUAL TOOL FILES (copy via FileExplorer before platform build)
----------------------------------------------------------------
  tools/node-attacker.src    ->  /Corvid/tools/node-attacker.src
  tools/node-router.src      ->  /Corvid/tools/node-router.src
  tools/csniffer.src         ->  /Corvid/tools/csniffer.src
  tools/rot.src              ->  /Corvid/tools/rot.src
  tools/chainsaw/            ->  /Corvid/tools/chainsaw/   (whole folder)


FIRST LAUNCH
------------
  /Corvid/corvid
  platform build    <- compiles node-attacker and node-router


PORT FORWARDING
---------------
Port 1222 must be forwarded on your Nest router for flock/roost to work.


MANUAL INSTALL (alternative)
-----------------------------
Copy corvid2_ingame/source/ -> /Corvid/source/
Then in-game: build /Corvid/source/main /Corvid/corvid
