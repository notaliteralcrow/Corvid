CORVID — INSTALLATION
=========================

STEP 1 — Run installer scripts 0-8 in Code Editor
---------------------------------------------------
Run each in order. For each: open in Code Editor, Save, Build, Run.
Wait for completion before starting the next.

  installer0.src  through  installer8.src  — write all source and tool files
  installer9.src  — builds /Corvid/corvid binary

All source files, tools, and chainsaw data are written automatically.
No manual file copying required.


STEP 2 — Launch Corvid and build tool binaries
-----------------------------------------------
  /Corvid/corvid
  platform build    <- compiles node-attacker and node-router
  chainsaw build    <- compiles standalone chainsaw tool


PORT FORWARDING
---------------
Port 1222 must be forwarded on your Nest router for flock/roost to work.
